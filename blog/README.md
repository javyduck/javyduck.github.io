# Blog

Static blog served at `https://javyduck.github.io/blog/`, generated by
`_src/build.py` (the `_src/` directory is excluded from the published site by
GitHub Pages' Jekyll, but stays in git).

## Adding a post

1. Write the post:
   - **Markdown**: put `index.md` (plus any images) in `_src/content/<slug>/`.
     Inline math `$...$`, display math `$$...$$`, code blocks, and images all
     work. Images referenced by relative path are copied next to the output.
   - **LaTeX chapter** (from the `math` repo): point `source` at the
     chapter's `src/chapter.tex`. TikZ figures are not converted
     automatically — put a hand-made SVG at
     `posts/<slug>/figures/<figure-label>.svg` (label = the part of the
     `fig:` id after `fig:`) and the build injects it.
2. Register it in `_src/posts.json` (slug, title, date, category,
   description, format, source; `chapter` + `bibliography` for LaTeX posts).
   Categories are defined at the top of the same file.
3. Rebuild:

   ```bash
   cd blog/_src && python3 build.py
   ```

   Requires pandoc (`~/.local/pandoc/pandoc-3.6.4-arm64/bin/pandoc` is picked
   up automatically; or pass `--pandoc /path/to/pandoc`).
4. Preview locally:

   ```bash
   cd <repo root> && python3 -m http.server 8123
   # open http://localhost:8123/blog/
   ```

Everything under `posts/` and `index.html` is generated — edit templates in
`_src/templates/`, styles in `assets/blog.css`, then re-run the build.

View counts come from busuanzi (free, no account); they only count on the
real domain, not localhost.
